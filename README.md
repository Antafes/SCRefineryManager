# SCRefineryManager

This is a small management tool for Star Citizen refinery orders.
It allows you to check on how much money you have made by refining and selling the mined materials.

The download comes with an executable for Windows.
Mac and Linux users would need to install Java 25 or higher.

! Currently only available in English and German !

## Usage

To add a new refinery order, click on file -> new.
A new window will open where you can enter the cost for the order and which materials are being refined.
The amount field is in cSCU as listed in the refinery window ingame.
You can also add inert materials, as you have to sell that stuff to get back empty ore pods.
After clicking on save, the order will be stored and shown in the main window.
